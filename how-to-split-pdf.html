How to Split a PDF Into Separate Files

Splitting a PDF means extracting specific pages or page ranges into separate downloadable files. PDFyre lets you define multiple ranges at once and download each as an independent PDF — completely free, in your browser, with no upload required.

When Do You Need to Split a PDF?

  • Sharing a specific chapter from a large book without the entire file
  • Extracting an invoice or certificate from a multi-document PDF
  • Separating sections of a report for different recipients
  • Removing unwanted pages from a scanned document
  • Breaking a large file into smaller pieces for email attachment limits

How to Split a PDF with PDFyre

1

Open the Split PDF tool

Go to PDFyre.com and click Split PDF on the homepage.

2

Select your PDF

Click the drop zone to load your PDF. PDFyre reads the total page count and displays it immediately.

3

Define your page range

Enter the from page and to page numbers. For example: pages 5 to 20. You can also name the output file.

4

Add more ranges if needed

Click Add another range to define multiple extractions from the same file. Each range produces a separate downloadable PDF.

5

Click Split PDF

PDFyre extracts each range and produces individual PDF files. Original page quality is preserved exactly.

6

Download your files

A download button appears for each output file. Click each one to save.

💡 Pro tip: You can extract multiple non-consecutive ranges in one operation. For example: pages 1–10, pages 25–30, and pages 45–60 all in a single split run, each producing a separate file.

What is Preserved When Splitting

PDFyre copies pages directly from the source PDF using pdf-lib. This means:

  • Full original image quality — no re-rendering
  • Any existing searchable text layers are preserved
  • Page dimensions remain unchanged
  • No watermarks are added
