PDFyre is a free, browser-based PDF tool suite. Every single tool — OCR, Merge, Split, Compress — runs entirely inside your browser. Your files never leave your device. Not even for a millisecond.

Why PDFyre Exists

Most online PDF tools upload your documents to remote servers. This is a problem. PDFs often contain sensitive information — medical records, legal documents, academic papers, financial statements, personal IDs. Uploading them to an unknown server means trusting a stranger with your data.

PDFyre was built on a simple principle: your files are yours. All processing happens on your own CPU, in your own browser, with no network requests containing your data.

What PDFyre Can Do

🔥

OCR PDF

Make scanned PDFs searchable and copy-pasteable. Word-level text alignment. multiple languages. 5 quality modes.

🔀

Merge PDF

Combine multiple PDFs into one. Original quality preserved. No file size limits. No watermarks.

✂️

Split PDF

Extract page ranges into separate files. Define multiple ranges. Name each output file.

📦

Compress PDF

Reduce file size with 3 compression levels. Best for scanned and image-heavy PDFs.

Built With

📄
PDF.js
Mozilla's PDF renderer. Renders PDF pages to canvas in-browser.
🔍
Tesseract.js
Browser port of the Tesseract OCR engine. Supports multiple languages via LSTM neural networks.
🛠️
pdf-lib
JavaScript PDF creation and modification library. Builds the output PDF with embedded text layers.
🗄️
IndexedDB
Browser-native storage for buffering large file processing without RAM crashes.
